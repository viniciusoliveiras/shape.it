## How to run

</br >

Clone this repository
```bash
git clone https://github.com/viniciusoliveiras/shape.it.git
```

 Access the project folder on your terminal/cmd
```bash
cd shape.it
```

Install the dependencies
```bash
 yarn
```

Run the application
```bash
yarn dev
```
